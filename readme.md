# Hang Man

January 28, 2016

##### By Molly Curtin &amp; Torrence Stratton

### Description

A web application where a user chooses letters until they've either filled the empty spaces or loses by making an entire Hangman.

* HTML
* CSS
* JavaScript
* jQuery
* Mocha
* Chai
